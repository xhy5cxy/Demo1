# 图片上传功能使用说明

## 后端接口

### 1. 图片上传接口
- **URL**: `POST /api/upload/image`
- **描述**: 上传普通图片到阿里云OSS
- **参数**: `file` (MultipartFile) - 图片文件
- **返回**: 图片URL地址
- **限制**: 只能上传图片文件，大小不超过5MB

### 2. 头像上传接口
- **URL**: `POST /api/upload/avatar`
- **描述**: 上传头像到阿里云OSS
- **参数**: `file` (MultipartFile) - 头像文件
- **返回**: 头像URL地址
- **限制**: 只能上传图片文件，大小不超过5MB

## 前端使用示例

### 基本图片上传
```vue
<template>
  <el-upload
    action="/api/upload/image"
    :on-success="handleSuccess"
    :before-upload="beforeUpload">
    <el-button type="primary">点击上传</el-button>
  </el-upload>
</template>

<script setup>
const handleSuccess = (response) => {
  console.log('上传成功:', response.data)
}

const beforeUpload = (file) => {
  const isImage = file.type.startsWith('image/')
  const isLt5M = file.size / 1024 / 1024 < 5
  
  if (!isImage) {
    ElMessage.error('只能上传图片文件!')
    return false
  }
  if (!isLt5M) {
    ElMessage.error('图片大小不能超过 5MB!')
    return false
  }
  return true
}
</script>
```

### 头像上传
```vue
<template>
  <el-upload
    class="avatar-uploader"
    action="/api/upload/avatar"
    :show-file-list="false"
    :on-success="handleAvatarSuccess">
    <img v-if="avatarUrl" :src="avatarUrl" class="avatar" />
    <el-icon v-else class="avatar-uploader-icon"><Plus /></el-icon>
  </el-upload>
</template>

<script setup>
import { ref } from 'vue'

const avatarUrl = ref('')

const handleAvatarSuccess = (response) => {
  if (response.code === 200) {
    avatarUrl.value = response.data
  }
}
</script>
```

## 文件存储结构

上传到阿里云OSS的文件会按照以下结构存储：
- 普通图片: `images/YYYYMMDD/{UUID}.jpg`
- 头像图片: `avatars/YYYYMMDD/{UUID}.jpg`

## 配置说明

### 后端配置 (application.yml)
```yaml
aliyun:
  oss:
    endpoint: your-endpoint
    access-key-id: your-access-key-id
    access-key-secret: your-access-key-secret
    bucket-name: your-bucket-name
spring:
  servlet:
    multipart:
      max-file-size: 10MB
      max-request-size: 100MB
```

### 跨域配置
已自动配置跨域支持，前端可以直接访问后端接口。

## 使用步骤

1. **配置阿里云OSS**: 在 `application.yml` 中填写你的阿里云OSS配置信息
2. **重启后端服务**: 使配置生效
3. **使用上传组件**: 在前端页面中使用 `<ImageUpload>` 组件或自定义上传逻辑
4. **处理返回结果**: 上传成功后会返回图片的URL地址

## 完整示例组件

项目中已提供完整的图片上传示例组件：
- 文件路径: `src/components/ImageUpload.vue`
- 包含功能：拖拽上传、点击上传、头像上传、上传预览
- 使用方法：直接在页面中引入该组件即可

## 注意事项

1. 确保阿里云OSS配置正确，包括endpoint、accessKey等
2. 文件大小限制为5MB，格式限制为图片类型
3. 上传的文件会自动重命名，避免文件名冲突
4. 返回的URL可以直接在浏览器中访问
5. 建议在生产环境中添加文件类型和大小的严格验证
