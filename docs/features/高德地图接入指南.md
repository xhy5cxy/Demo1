## 3. 高德地图URL Scheme说明

### 3.1 App跳转URL
```
amapuri://route/plan/?dlat=39.908823&dlon=116.397470&dname=天安门&dev=0&t=0
```

**参数说明：**
- `dlat`：目的地纬度（示例：39.908823）
- `dlon`：目的地经度（示例：116.397470）
- `dname`：目的地名称（示例：天安门）
- `dev`：是否偏移（0：不偏移，1：偏移）
- `t`：导航类型（0：驾车，1：公交，2：步行）

### 3.2 网页版导航URL
```
https://uri.amap.com/navigation?to=天安门&mode=car&src=etour
```

**参数说明：**
- `to`：目的地名称或坐标
- `mode`：导航模式（car：驾车，bus：公交，walk：步行）
- `src`：来源标识

## 4. 高德地图SDK完整接入指南

### 4.1 申请API Key

1. 访问[高德开放平台](https://lbs.amap.com/)
2. 注册开发者账号
3. 创建应用获取API Key

### 4.2 添加依赖

在`app/build.gradle.kts`中添加：

```kotlin
dependencies {
    // 高德地图定位SDK
    implementation 'com.amap.api:location:latest.version'
    
    // 高德地图3D地图SDK
    implementation 'com.amap.api:3dmap:latest.version'
    
    // 高德地图搜索SDK
    implementation 'com.amap.api:search:latest.version'
    
    // 高德地图导航SDK
    implementation 'com.amap.api:navi-3dmap:latest.version'
}
```

### 4.3 配置权限

在`AndroidManifest.xml`中添加：

```xml
<!-- 网络权限 -->
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />

<!-- 定位权限 -->
<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
<uses-permission android:name="android.permission.ACCESS_LOCATION_EXTRA_COMMANDS" />

<!-- 高德地图服务 -->
<service android:name="com.amap.api.location.APSService" />

<!-- 高德地图API Key -->
<meta-data
    android:name="com.amap.api.v2.apikey"
    android:value="你的API_KEY" />
```

### 4.4 初始化地图

```java
// 在Application中初始化
public class MyApplication extends Application {
    @Override
    public void onCreate() {
        super.onCreate();
        
        // 初始化高德地图
        AMapLocationClient.setApiKey("你的API_KEY");
        
        // 隐私政策合规
        MapsInitializer.updatePrivacyShow(this, true, true);
        MapsInitializer.updatePrivacyAgree(this, true);
    }
}
```

```java
private void openAmapWithFallback() {
    try {
        Intent intent = new Intent(Intent.ACTION_VIEW);
        intent.setData(Uri.parse("amapuri://route/plan/?dlat=39.908823&dlon=116.397470"));
        
        if (intent.resolveActivity(getPackageManager()) != null) {
            startActivity(intent);
        } else {
            // 高德地图未安装，使用网页版
            Intent webIntent = new Intent(Intent.ACTION_VIEW);
            webIntent.setData(Uri.parse("https://uri.amap.com/navigation?to=天安门"));
            startActivity(webIntent);
        }
    } catch (Exception e) {
        // 最终兜底方案
        Intent browserIntent = new Intent(Intent.ACTION_VIEW);
        browserIntent.setData(Uri.parse("https://www.amap.com/"));
        startActivity(browserIntent);
    }
}
```

## 8. 常见问题

### Q: 跳转后高德地图未打开？
A: 检查设备是否安装高德地图，或使用网页版作为备选方案

### Q: 坐标定位不准确？
A: 确保使用WGS84坐标系，高德地图支持GCJ-02坐标系

### Q: 导航参数无效？
A: 参考高德官方文档检查URL Scheme参数格式
