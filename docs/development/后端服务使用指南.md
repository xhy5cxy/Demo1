# 后端服务使用指南

## 🎯 服务信息
- **服务名称**: E旅游攻略系统后端服务
- **运行端口**: 8888
- **基础路径**: /api
- **API文档**: http://localhost:8888/api/doc.html

## 🚀 启动后端服务

### 方法一：使用批处理文件（推荐）
```bash
cd "d:\tour\etour"
.\start.bat
```

### 方法二：直接使用Maven命令
```bash
cd "d:\tour\etour"
mvn spring-boot:run
```

### 方法三：使用开发环境配置
```bash
cd "d:\tour\etour"
mvn spring-boot:run -Dspring-boot.run.profiles=dev
```

## ⏹️ 停止后端服务

### 方法一：Ctrl+C（推荐）
在运行后端服务的终端中按 `Ctrl + C`

### 方法二：使用任务管理器
1. 打开任务管理器
2. 找到Java进程或Maven进程
3. 结束相关进程

### 方法三：命令行关闭
```bash
# 查找Java进程
jps -l

# 结束Spring Boot应用进程
taskkill /f /pid <进程ID>
```

## 🛠️ 常见问题

### 端口被占用
```bash
# 查看端口占用情况
netstat -ano | findstr :8888

# 强制释放端口
taskkill /f /pid <占用进程ID>
```

### 服务启动失败
1. 检查Java环境：`java -version`
2. 检查Maven环境：`mvn -version`
3. 清理项目：`mvn clean`
4. 重新编译：`mvn compile`

### 网络连接问题
1. 检查防火墙设置
2. 确认端口8888已开放
3. 检查IP地址配置
4. 使用网络诊断工具测试
