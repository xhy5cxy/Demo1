## 🔧 技术栈

### 后端
- **框架**: Spring Boot 3.x
- **端口**: 8888
- **文档**: SpringDoc OpenAPI 3
- **数据库**: MySQL + MyBatis-Plus

### 前端 (安卓)
- **网络**: Retrofit 2.x + OkHttp 3.x
- **JSON**: Gson
- **架构**: MVVM + Dagger 2
- **测试**: 专用测试活动
